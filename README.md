Other languages coming soon!

# How to Use

Once opening the website, type some text into the textbox using your keyboard and click the speaker, then, it will play for you!

## Downloading?

Sadly, the Javascript speechSynthesis function doesn't allow users to download a file, however, you can use a recorder to do exactly thaat!
